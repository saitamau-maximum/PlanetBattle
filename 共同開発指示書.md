# 共同開発指示書

## 方針

- イメージは「にゃんこ大戦争」とマリオアクションと育成ゲームを混ぜた感じ。
- キャラを育てて戦わせる。（育成の手段は敵を倒した経験値でスキルを獲得したり攻撃力を上げたりできる。姿を変えたりはしない。）
- ステージを選択して、そこに無数の敵が出てくる。一定数倒したらステージクリア。それを繰り返して成長させる。ステージクリア後に経験値を各メーターやスキルに割り当てる。
- 同じステージを何度も繰り返せるようにする。
- 多少弱体化させてオート機能の実装
- 完全放置機能(ログアウトしてから再ログインまで時間で若干強化を入れる)
- 素材はアセットストアなどから無料の素材を入手。
- ゲームの目標としてはボスを倒せるようになるまで育成して戦わせる。
- マリオのようにキャラをボスまで連れていくのではなく、防衛ゲームにしてボスを迎え撃つ。
- 耐えきったらゲームクリア
- 基本一本道で敵を迎撃する
- 敵はランダム沸きで時間を計ってフェーズを移行する。
- コイン(戦闘内でのみ使える砲台の設置に使う通貨)。時間経過または敵を倒せばもらえる。
- プレイヤーはHPでダメージ管理して0になると数秒経ってから復活
- 敵はボスキャラ（最終フェーズに出てくる）のみHPゲージ表示。素材は敵キャラを大きくしたもの。
- 敵は画面内の茂みから出てくる
- 敵（動物）の攻撃は接触・敵（人型）の攻撃は弓矢
- タワーディフェンス(味方の固定砲台、あと床に罠設置)

## 設定
- プレイヤーは宇宙飛行士で、未知の惑星に不時着。そこに住む動物たちと戦いながら資源を集めて脱出を目指す。

## アイデア

- ダメージバーは8分割にして1/8経るごとに画面を更新（そうすれば処理が軽くなる）
- 条件に応じて攻撃力・防御力が上がったりする宝石の導入。宝石は何個かしか装備できない。宝石はレベルを付与する(1~5)。宝石は+/-で計算。スキルより先にこっち計算。
- 条件に応じて攻撃力・防御力が上がったりするスキルの導入。スキルは一度手に入れるとずっと使える。スキルは何個でも使えて上限はない。スキルは×/%で計算。
- ガチャの導入。経験値とは別にゴールドをもらえて、それでガチャを回せる。それでスキルや宝石をもらえる
