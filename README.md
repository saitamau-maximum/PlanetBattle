# NewGame2

ゲーム開発班の新しいゲーム開発用のリポジトリです。

# 操作方法
- タイプA
  - **A,D** : 左右移動
  - **Space** : ジャンプ
  - **左クリック** : 攻撃1、建物の建築
  - **右クリック** : 攻撃2
  - **F** : 攻撃モード・建築モード切り替え
  - **1 2 3 4** : スロット選択

- タイプB
  - **← →** : 左右移動
  - **↑** : ジャンプ
  - **C** : 攻撃1、構造物の建築
  - **X** : 攻撃2
  - **F** : 攻撃モード・建築モード切り替え
  - **1 2 3 4** : スロット選択
